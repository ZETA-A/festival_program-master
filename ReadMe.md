# GSC(Google Sheet Counting) 프로그램
## 만든 사유 : 왜 만들었을까

### 제작자 : 제타
### 사용법 :

1. 프로그램을 다운로드하고, GSC.exe를 실행한다.
2. 설정을 눌러 설정한 후 적용을 누른다.
3. 설정 창을 닫고 첨부된 엑셀파일을 구글 드라이브에 올린다.
4. 업데이트를 눌러 구글 시트의 정보를 받아온다.
